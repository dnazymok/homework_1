# Light IT - Homework 1

1. Clone project

`git clone https://github.com/dnazymok/homework_1.git`

2. Change directory 

`cd homework_1`

3. Create virtual environment

`virtualenv venv`

4. Activate venv

`. venv/bin/activate`

5. Install Lektor

`pip install Lektor`

6. Change directory 

`cd cheatsheet`

7. Serve project

`lektor serve`

8. Open 127.0.0.1:5000 in your browser
